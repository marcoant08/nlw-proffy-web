#### O server desta aplicação está disponível em:

#### Para executar esta aplicação:
 1. `git clone https://github.com/marcoant08/nlw-proffy-web`
 2. `cd nlw-proffy-web`
 3. `npm install`
 4. `npm run start`
 
 ###### A aplicação rodará na porta 3000
